# Indocurry's Digital Garden 

A curated collection of high-signal YouTube channels for learning, technology, and personal growth. Hand-picked to separate the signal from the noise.

## Live Site
**[Visit my Digital Garden](https://sidshot.github.io/Indocurry-recommendations-YT/)**

##  Features
*   ** Indocurry Branding:** Spicy Orange accents (`#ea580c`) & personalized handwritten typography.
*   ** Clean Design:** A premium "Whitish Grey" light theme for a calm reading experience.
*   ** Interactive:** fast search, category filters, and a "Random Channel" button for serendipity.
*   ** Responsive:** Glassmorphism effects that look great on desktop and mobile.

##  Built With
*   **HTML5 & CSS3** (Vanilla, no frameworks)
*   **JavaScript** (Embedded data for performance)
*   **Fonts:** [Outfit](https://fonts.google.com/specimen/Outfit) & [Patrick Hand](https://fonts.google.com/specimen/Patrick+Hand)
*   **Libraries:** `vanilla-tilt.js` for 3D card effects.
